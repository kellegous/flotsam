# Flotsam: bits of code and content I throw out to sea.

Sometimes I post articles at [kellegous.com](https://kellegous.com) and those posts contain code examples that I absolutely to not intend to maintain over time. But as a snapshot in time, they can be useful to demonstrate a concept or technique. This is where I keep those code examples. I may also put other bits in here that do not end up in articles. It's really just bits that slide off the deck of the ship over time.
